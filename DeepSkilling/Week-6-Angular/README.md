# Week 6 - Angular Framework

## Overview
This directory contains the hands-on exercises and project modules completed for **Week 6 - Angular Framework** as part of the **Cognizant Digital Nurture 5.0 Deepskilling Program**.

---

## Topics Covered

### 1. Angular Fundamentals
- Angular CLI and Workspace Setup
- TypeScript Integration & Component Architecture
- Module Organization (`AppModule`, Feature Modules)

### 2. Data Binding & Directives
- Interpolation (`{{ }}`), Property Binding (`[]`), Event Binding (`()`), and Two-Way Data Binding (`[(ngModel)]`)
- Structural Directives (`*ngIf`, `*ngFor`, `*ngSwitch`)
- Attribute Directives (`ngClass`, `ngStyle`)

### 3. Components & Services
- Parent-Child Component Communication (`@Input()`, `@Output()`, `EventEmitter`)
- Dependency Injection & Service Providers (`@Injectable()`)
- Shared Data Services & State Management

### 4. Routing & Navigation
- Angular Router Configuration (`RouterModule.forRoot()`)
- Route Parameters (`ActivatedRoute`) & Navigation Services (`Router`)
- Route Guards (`CanActivate`) for Access Control

### 5. Forms & HTTP Client
- Template-driven Forms & Reactive Forms (`FormBuilder`, `Validators`)
- Asynchronous Data Fetching using `HttpClientModule`
- Observables, Promises, and RxJS Operators (`map`, `filter`, `catchError`)
