# Module 1 - Frontend Development

## Overview
This directory contains the hands-on code exercises, practice labs, and study resources for **Module 1 - Frontend Development** as part of the **Cognizant Digital Nurture 5.0 Upskilling Program**.

---

## Technologies & Topics Covered

### 1. HTML5
- Semantic Tags (`<header>`, `<nav>`, `<article>`, `<footer>`)
- Form Elements & Validation Attributes
- Audio, Video, and Canvas Integration

### 2. CSS3
- Selectors, Box Model, and Positioning
- Flexbox & CSS Grid Layouts
- Animations, Transitions, and Responsive Media Queries

### 3. JavaScript
- ES6+ Syntax (Arrow Functions, Promises, Async/Await)
- DOM Manipulation & Event Handling
- JSON Parsing and Fetch API

### 4. jQuery
- Selectors & DOM Traversal
- Event Binding & Effects (Fade, Slide)
- AJAX Requests

### 5. Bootstrap 5
- Grid System & Breakpoints
- Responsive Components (Navbar, Cards, Modals, Buttons)
- Utility Classes
