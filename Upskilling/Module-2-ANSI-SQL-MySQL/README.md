# Module 2 - ANSI SQL & MySQL

## Overview
This directory contains SQL scripts, query exercises, notes, and assessment verification for **Module 2 - ANSI SQL & MySQL** as part of the **Cognizant Digital Nurture 5.0 Upskilling Program**.

---

## Topics Covered

- **Database Fundamentals**: DDL (CREATE, ALTER, DROP) and DML (INSERT, UPDATE, DELETE) statements
- **Data Querying**: `SELECT`, `WHERE`, `ORDER BY`, `GROUP BY`, `HAVING`
- **Joins**: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, and Self Joins
- **Subqueries & Aggregate Functions**: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`, Subqueries, and Nested Queries
- **Constraints & Indexes**: Primary Key, Foreign Key, Unique, Not Null, Check, and Indexing for optimization

---

## Files Included

- `SQL-Queries.sql` - Practical SQL query scripts and hands-on exercises
- `SQL-Notes.docx` - Comprehensive notes and theoretical concepts
- `SQL-Quiz-Screenshots.pdf` - Assessment completion verification
