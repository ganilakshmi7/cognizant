# Module 3 - C# & ADO.NET Programming

## Overview
This directory contains C# code implementations, ADO.NET database connectivity scripts, notes, and assessment verification for **Module 3 - C# & ADO.NET** as part of the **Cognizant Digital Nurture 5.0 Upskilling Program**.

---

## Topics Covered

- **C# Language Core**: Data types, OOP Principles (Encapsulation, Inheritance, Polymorphism, Abstraction), Interfaces, Exception Handling
- **Collections & LINQ**: Lists, Dictionaries, Lambda Expressions, and LINQ Queries
- **ADO.NET Architecture**:
  - Connected Architecture (`SqlConnection`, `SqlCommand`, `SqlDataReader`)
  - Disconnected Architecture (`SqlDataAdapter`, `DataSet`, `DataTable`)
- **CRUD Operations**: Database connection management, executing stored procedures, and parameterized queries
